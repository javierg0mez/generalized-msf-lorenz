# Generalized MSF for the Lorenz System

This repository contains the numerical implementation accompanying the work on a generalized Master Stability Function (MSF) for more general diffusive couplings.

The results were presented in the poster

> **A Generalization of the Master Stability Function for More General Diffusive Couplings**

at the **Ibero-American School on Networks and Complex Systems (ENREDANDO 2026)**, Viçosa, Minas Gerais, Brazil.

## Contents

- Python notebooks,
- Numerical simulations,
- Figures used in the poster.
